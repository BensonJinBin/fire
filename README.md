# Fire计算器微信小程序

## 项目简介
这是一个基于微信小程序平台开发的Fire（财务独立，提前退休）计算器，帮助用户规划财务独立之路。

## 功能特色
- 年初本金、年收入、年支出、投资回报率、通胀率等参数设置
- 考虑通胀影响的复利计算
- 详细的年度资金变化表格
- 智能分析与建议
- 丰富的Fire相关知识介绍

## 计算公式
- 年末本金 = 年初本金 - 年消费 + 年利息 + 年收入
- 年利息 = 年初本金 × 年投资回报率
- 年消费会随着通胀率逐年增长

## 使用方法
1. 在微信开发者工具中打开此项目
2. 填写各项财务参数
3. 点击"开始计算"查看结果
4. 根据分析建议调整财务规划
